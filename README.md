# 🛒 E-Commerce Mobile App

Aplikasi **E-Commerce berbasis mobile** menggunakan **Expo Go (React Native)** sebagai frontend dan **Node.js (Express.js)** sebagai backend.  
Aplikasi ini memungkinkan pengguna melakukan proses jual beli online mulai dari melihat produk, menambahkan ke keranjang, hingga melakukan pemesanan.

Project ini dibuat sebagai latihan dan pengembangan **fullstack mobile development**.

---

## ✨ Fitur

### 👤 User
- Register & Login
- Lihat daftar produk
- Detail produk
- Keranjang belanja
- Checkout
- Riwayat pesanan

### 🛠️ Admin
- Tambah, edit, hapus produk
- Manajemen stok
- Manajemen pesanan

---

## 🧱 Tech Stack

### 📱 Mobile App
- Expo Go
- React Native
- Axios
- React Navigation

### 🌐 Backend
- Node.js
- Express.js
- REST API
- JWT Authentication

### 🗄️ Database
- MongoDB / MySQL / PostgreSQL
